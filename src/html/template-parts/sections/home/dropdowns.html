Frequently asked questions

Does the system offer only one content option or several?

Users can generate content 3 times within one order but choose and pay only for the one they like.

Does the user have to pay immediately?

No, the system shows the textual and visual results as a preview, and the user chooses the best option.

Can I change the content request in one order?

Yes, you can create content requests thrice within the same session. This way, you can create 3 separate sessions within one user account without any additional fees.
